## Android‑приложение прогноза погоды на Kotlin.
### Функционал
* Показывает текущую погоду, почасовой прогноз на 24 часа и ежедневный на неделю (температуры, осадки, влажность, ветер, давление, видимость, рассвет/закат). 
* Поддерживает определение местоположения по GPS с настройкой точности/таймаута, поиск городов по названию, список избранных городов, историю поиска и кеш последней локации.
* В настройках — единицы измерения, язык интерфейса (RU/EN), точность GPS, таймаут GPS и период автообновления.

### Технологии и библиотеки: 
Jetpack Compose, Navigation Compose, Kotlin Coroutines, Retrofit, OkHttp, Room для избранного, DataStore для настроек и истории Hilt для DI.
